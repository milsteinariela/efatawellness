# Efata Embodied Wellness Website

Official website for Efata Embodied Wellness in the Riviera Maya. This site showcases integrative psychotherapy, somatic bodywork, and ancestral ceremonies offered by Ariela Milstein and Kinsey Williams.

## Features
* **Responsive Design:** Fully adaptable layout that works on desktops, tablets, and mobile phones.
* **Mobile Navigation:** Custom hamburger menu for smaller screens.
* **Bilingual Support:** Integrated Google Translate widget to toggle between English and Spanish.
* **Services Showcase:** Detailed sections for Somatic Bodywork, Psychotherapy, Temazcal, Psilocybin, and Corporate Wellness.

## How to View
To view this website live, enable **GitHub Pages** in the repository settings, or simply download the files and open `index.html` in your browser.
